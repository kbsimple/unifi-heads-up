# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-04-19
**Phases:** 1–4
**Total Plans:** 16

## Overview

First complete release of the Unifi Network Dashboard — a Next.js web application for monitoring home network traffic and managing firewall rules via the Unifi Site Manager Proxy. Delivered authentication, real-time device visibility, firewall toggle controls, device groups, and 24-hour traffic history.

## Milestone Summary

**Key Decisions:**
- Site Manager Proxy over VPN — simpler setup, works behind CGNAT
- Next.js + Vercel — full-stack framework with zero-config deployment
- Polling over WebSocket — 5-min averages don't need real-time streaming
- Toggle existing rules only — lower complexity, user already has rules configured
- localStorage for groups — no database needed for v1 single-user family app
- useRef (not useState) for traffic history accumulation — avoids re-renders on every SWR poll
- Vitest + RTL — recommended for Next.js 15+ test infrastructure

**Issues Resolved:**
- JWT lazy key encoding workaround for Vitest env var timing
- jose mocking in jsdom environment
- SWR double mockReturnValueOnce for TrafficHistoryProvider + ClientListInner
- App routing fixes and API error handling improvements (added in quick tasks)
