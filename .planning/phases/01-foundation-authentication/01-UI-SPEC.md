## Layout Specifications

### Login Page
- **Focal point:** Sign in button — the only colored action element draws the eye
- **Visual priority:** App title > form fields > Sign in button
- Centered card: `max-w-md` (448px width)
- Vertical centering: `min-h-screen` with flexbox center
- Card padding: `p-6` (24px)
- Form field gap: `space-y-4` (16px between fields)

### Navigation Bar (App Shell)
- Height: 64px (4xl)
- Horizontal padding: 24px (lg)
- Background: zinc-900 with bottom border zinc-800
- App name: left-aligned, heading size (20px, semibold)
- Logout button: right-aligned, ghost variant

### Toast Notifications
- Position: bottom-right
- Auto-dismiss: 5 seconds
- Max visible: 3 stacked
- Error variant: destructive styling
