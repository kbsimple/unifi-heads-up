      ```typescript
      }
      ```
    - Provider component: `TrafficHistoryProvider`
    - Hook: `useTrafficHistory()` throws if used outside provider
    - Implementation:
      - Use useRef to store minuteSamples array (raw 60-second samples)
      - Use useRef to store hourlyBuckets array (aggregated hourly averages)
      - Use useSWR hook to subscribe to '/api/clients' data
      - On each successful SWR update (every 60s per DEVI-05):
        - Capture a MinuteSample with timestamp, totals, and per-client data
        - Add to minuteSamples array
        - If minuteSamples for current hour has >= 60 entries, aggregate into hourly bucket:
          - hourStart: floor timestamp to hour
          - avgDownload: average of all minute samples in that hour
          - avgUpload: average of all minute samples in that hour
          - sampleCount: number of samples
        - Keep only last 24 hourly buckets: `hourlyBuckets.slice(-24)`
      - Store per-client history in a Map: clientId -> HourlySample[]
    - Per D-18: Data stored in React state/ref, lost on page refresh (acceptable for MVP)

    **Hourly aggregation logic:**
    ```typescript
    function aggregateToHourly(samples: MinuteSample[]): HourlySample {
      const avgDownload = samples.reduce((s, m) => s + m.totalDownload, 0) / samples.length
      const avgUpload = samples.reduce((s, m) => s + m.totalUpload, 0) / samples.length
      return {
        hourStart: Math.floor(samples[0].timestamp / 3600000) * 3600000,
        avgDownload,
        avgUpload,
        sampleCount: samples.length
      }
    }
    ```
