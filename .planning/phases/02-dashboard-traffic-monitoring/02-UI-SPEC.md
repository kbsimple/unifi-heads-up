## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, card internal gaps |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, main content padding |
| xl | 32px | Layout gaps, section spacing |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Navigation bar height, page-level spacing |

Exceptions:
- Badge padding: 4px horizontal, 2px vertical (shadcn default)
- Traffic badge: 8px horizontal padding for better visibility

## Color

| Status | Background | Text | Usage |
|--------|------------|------|-------|

| Low | green-500/20 | green-400 | Device with 1-10 Mbps traffic |
| Medium | yellow-500/20 | yellow-400 | Device with 10-100 Mbps traffic |
| High | red-500/20 | red-400 | Device with > 100 Mbps traffic |

## Layout Specifications

### Dashboard Page Structure

```
+------------------------------------------+
| Navigation Bar (64px height)             |
| [Unifi Dashboard]              [Sign out]|
+------------------------------------------+
| Main Content (p-6 = 24px padding)        |
|                                          |
| [Section Title: Network Clients]         |
| [Last updated: 2m ago]  [Refresh button] |
|                                          |
| +--------------------------------------+ |
| | Client Cards (mobile) / Table (desk) | |
| | ...                                  | |
| +--------------------------------------+ |
+------------------------------------------+
```

### Navigation Bar
- Height: 64px (3xl)
- Background: zinc-900 with zinc-800 bottom border
- App name: left-aligned, 20px semibold, zinc-100
- Logout button: right-aligned, ghost variant

### Client Card (Mobile View)

```
+----------------------------------------+
| [Device Name]              [Badge]     |
| [IP Address] - [MAC Address]           |
|----------------------------------------|
| Last active: 5m ago                    |
+----------------------------------------+
```

- Card background: zinc-900
- Card border: zinc-800
- Card padding: p-4 (16px)
- Card radius: rounded-lg
- Card gap: space-y-3 (12px) between cards
- Device name: font-medium, zinc-100
- IP/MAC: text-sm, zinc-400
- Last active: text-xs, zinc-500
- Border-top divider: zinc-800

### Client Table (Desktop View)

| Column | Width | Alignment |
|--------|-------|-----------|
| Device Name | flex-1 | left |
| IP Address | 140px | left |
| MAC Address | 160px | left |
| Status | 100px | center |
| Last Active | 120px | right |

- Table header: text-xs uppercase, zinc-500
- Table row: 48px height, border-b zinc-800
- Row hover: zinc-800/50 background

### Traffic Badge Component

- Variant: outline
- Size: default
- Padding: px-2 py-0.5 (8px horizontal, 2px vertical)
- Font: text-xs font-medium
- Radius: rounded-md
- Colors per status table above

### Last Updated Component

- Position: inline with section title
- Text: text-sm, zinc-400
- Loading indicator: RefreshCw icon with animate-spin (12px icon)
- Format: "Last updated: {time}"

### Error/Offline State (Alert)

- Variant: destructive
- Icon: AlertCircle (16px)
- Title: "Unable to reach network service"
- Description: "Please check your connection and try again."
- Action: "Retry" button (outline variant, small)

### Empty State

- Container: centered, p-8
- Icon: WifiOff (32px, zinc-500)
- Title: "No devices found"
- Description: text-zinc-400

## Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| < 768px (mobile) | Card list, single column |
| >= 768px (tablet+) | Table view with all columns |
| >= 1024px (desktop) | Table view, wider columns |

**Card-to-table switch:**
```tsx
<div className="md:hidden space-y-3">
  {/* Card list for mobile */}
</div>
<div className="hidden md:block">
  {/* Table for tablet/desktop */}
</div>
```
